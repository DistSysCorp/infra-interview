# infra-interview（基础架构面试）

Interview data structures and algorithms for infra programmers.

本仓库主要辑录 infra 程序员常见面试题目：数据结构、算法、语言（主要是 C++）。
欢迎大家在 issue 里补充你遇到的有趣的题目，然后我增加题解放到仓库里。当然，如果你直接提 PR 更好。
